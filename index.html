Color Mapper

This tool takes as inputs a list of Match Candidates and Input Values which will result in the set of appropriate color mappings. This will be used to determine how to take an arbitrary set of color values and have them conform to a finite set of colors. The closeness of the colors is determined by computing the Euclidean distance of each RGB value in the Match Candidates for a specific Input Value. The sorting is determined by sorting based on the luminosity of each RGB-value.

Match Candidates

Here, you will need to specify the set of colors that we will consider as match candidates. This means, that whatever is specified in Input Values will need be matched to one of the values specified here. A sample row of input will look something like: Sea Buckthorn:#f89728 or #f89728. The portion of the input is optional, though it will be left blank in the table results.

Upon changing this input, the results will be populated below. Both Match Candidates and Input Values need to be completed before producing a result.

Input Values

Similarly to Match Candidates, you will need to provide a list of properly formatted color inputs. These will be an arbitrary set of values that will be matched to one of the specified match candidates. The same formatting rules as Match Candidates needs to be enforced, however, the name can be excluded since it will not be displayed in the results.

Upon changing this input, the results will be populated below. Both Match Candidates and Input Values need to be completed before producing a result.
